УПРЗА — расчёт рассеивания загрязнения воздуха (МРР-2017)

      
Заполните источники выбросов (трубы). «Карта» — суммарное поле
        максимальных концентраций с изолинией ПДК; «Профиль» — по оси факела.
        Для госэкспертизы нужна аттестованная УПРЗА — данные можно выгрузить кнопкой ниже.
