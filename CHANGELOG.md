# История версий

## 0.4.0 — 2026-07-02

- **ИИ-слой** (`ecodoc ai`): автообнаружение локальных моделей (Ollama —
  используются все найденные chat-модели по цепочке fallback, LM Studio)
  и 13 внешних провайдеров (Anthropic, OpenAI, OpenRouter, DeepSeek, Gemini,
  Groq, Mistral, xAI, Together, VseGPT, ProxyAPI, GigaChat, YandexGPT);
  ключи — только через переменные окружения.
- **ИИ-анализ входящих** (`intake --ai`): извлечение реквизитов, актов
  утилизации, протоколов КХА/биотеста с дословной цитатой-источником для
  каждого значения; конфликты не перезаписываются, а показываются.
- **Мульти-организации × площадки** (`org add/list`, `site add`,
  `--org/--site` во всех командах): рабочее пространство на диске,
  реквизиты организации наследуются всеми её площадками.
- **Приём документов** (`intake`): реестр принятых файлов (sha1, дата),
  отчёт «что распознано / чего не хватает / какие документы ещё нужны»
  по каждой форме.
- **Сторож форм** (`watch check`): слежение за страницами приказов, ставок
  НВОС, ЛКПП, региональных кадастров (СПб/ЛО) и локальной папкой XSD.
- **Свой расчёт рассеивания** (`dispersion`) по МРР-2017: точечные
  источники, cм/xм/uм, профиль по оси, доля ПДК — см. docs/УПРЗА_анализ.md.
- **PDF-экспорт** (`pdf`, `generate --pdf`): MS Office COM → LibreOffice →
  текстовый снимок PyMuPDF.
- Каркасы контура «Разработка»: ПНООЛР, расчёт класса опасности, НМУ,
  ПЛАРН, ДВОС, инвентаризация выбросов, программа ПЭК, ТУ.
- Индекс эталонных документов (`data/etalon_index.json`) для верификации
  генераторов и обучения ИИ-анализа — локальный файл, в репозиторий
  не входит.
- Автоустановщик `install.bat`/`install.ps1`, пакетирование `pyproject.toml`
  (команда `ecodoc`), UTF-8 вывод в консоль Windows.

## 0.3.0

- 7 полных форм отчётности (XML + Excel): декларация НВОС (с расчётом
  платы), движение отходов, ПЭК, кадастр СПб/ЛО, 2-ТП (отходы/воздух),
  отчётность III категории/МСП; каркас 2-ТП (водхоз).
- Паспорта отходов I–IV класса из ФККО; каркасы томов НДВ/НДС/СЗЗ
  с импортом из «Эколога».
- Календарь подачи + чек-лист документов; парсер входящих
  (PDF/DOCX/DOC/JPG + OCR) с провенансом.
